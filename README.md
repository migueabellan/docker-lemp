# Hello World

> How to run php "Hello world" with Docker CE

## Prerequisites

Docker installation is required, see the [official installation docs](https://docs.docker.com/install/).

### Download using git

```sh
$ git clone https://github.com/migueabellan/docker-php.git
```

### Run container

```sh
$ docker-compose up -d
```

### Access project

```sh
http://localhost:8080
```
